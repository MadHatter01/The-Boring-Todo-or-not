## ✍🏻 The Boring TODO ever

Manage your tasks in the terminal!
With this **CLI-based task tracker**, you can:  
➕ Add tasks  
🗑️ Delete tasks  
📋 List tasks  
🎯 Mark tasks as completed  

Built using the **ultra-cool SQLAlchemy ORM** and **SQLite**
